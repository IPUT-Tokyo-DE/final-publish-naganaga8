# ここにゲームのタイトルを記述する
サーチandイート
## ゲームの内容
STARTシーンでSTARTボタンを押すとメインシーンへ移行する
上から計10種類の果物が落ちてくる
黄色いキューブをWASDキーor上下左右キーで上下左右に動かして果物の上に合わせると果物が消えスコアが加算される
右上に表示されている乱数に対応した果物しか消せない
乱数は5秒おきに変化する
スコアが200点以上でクリア、以下で失敗（乱数は右下に表示される）
クリアするとOKシーンへ移行、失敗するとOUTシーンへ移行する
to STARTボタンを押すとSTARTシーンへ移行する
